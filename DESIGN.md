# DESIGN — 깨! 앱 디자인 시스템

이 앱의 시각 언어를 정의합니다. 팔레트는 업로드된 **Super Mario Galaxy 2 커버에서 색을 추출·정제**해 도출했습니다. `SPEC.md`(기능)와 짝을 이루는 시각 단일 출처입니다.

> ⚠️ 저작권 — **닌텐도의 캐릭터·로고·이미지 자산은 사용하지 않습니다.** "요시 / 마리오"는 카피(문구)상의 컨셉 참조일 뿐, 실제 아트를 복제하지 않습니다. 알·별·코인 등은 팔레트만 차용한 **오리지널·추상 도형**으로 직접 그립니다. 이미지에서 가져오는 건 색뿐입니다.

---

## 1. 디자인 방향

**"어둠을 깨고 빛으로" — 갤럭시 어드벤처.** 밝고 경쾌한 게임 UI이되, 에베소서 5:8(어둠→빛)의 아크를 **색으로** 표현한다.

- **어둠(잠듦)**: 공동 카운터·잠든 요시의 알·부화 직전은 **스페이스 네이비**의 깊은 밤에 둔다.
- **빛(깨어남)**: 평상시 앱은 **밝은 하늘**, 알이 깨지는 순간 화면이 **갤럭시 하늘 + 별빛**으로 터진다.
- 톤: 친근하고 또렷한 게임 UI — 굵은 외곽선 + 하드 그림자(촉각적·장난스러움) + 밝은 하늘 그라데이션 + 별/코인 글로우.

---

## 2. 컬러 팔레트 (커버에서 추출·정제)

UI 대비를 위해 추출값을 약간 보정했습니다(원시값 → 보정값 표기).

| 토큰 | HEX | 추출 원본 | 용도 |
|---|---|---|---|
| `--sky` | `#3ABCE4` | 동일(32%) | **주 브랜드** 하늘 시안. 강조·하늘 그라데이션 |
| `--sky-deep` | `#2454C0` | `#204EB4` | 코발트(오버롤). 보조 강조·링크 |
| `--sky-soft` | `#EAF6FF` | 파생 | 앱 배경·옅은 면 |
| `--cloud` | `#FFFFFF` | 동일 | 카드·표면 |
| `--cloud-soft` | `#F3FAFE` | 파생 | 보조 표면 |
| `--star` | `#F6C92E` | `#F4D824` | **코인·보상·핵심 강조**(골드) |
| `--star-deep` | `#D0A527` | 동일 | 골드 그림자 |
| `--yoshi` | `#2E9E3F` | `#238133` | **성공·진행·팀리더**(그린) |
| `--yoshi-deep` | `#1E7A33` | 파생 | 그린 그림자 |
| `--mario` | `#E5392C` | `#9D413C` | **경고·에너지·관리자**(레드) |
| `--mario-deep` | `#B82B20` | 파생 | 레드 그림자 |
| `--space` | `#14224D` | 딥블루 계열 | **어둠**·본문 텍스트·다크 표면 |
| `--space-2` | `#0E1A3A` | 파생 | 더 깊은 스페이스(그라데이션) |
| `--ink-soft` | `#51618C` | 파생 | 보조 텍스트 |

**대비 주의(접근성):**
- 골드(`--star`)·시안(`--sky`) 위 작은 글씨는 대비가 낮음 → **면(fill)으로** 쓰고, 그 위 텍스트는 `--space`(진한색)로.
- 다크 표면(`--space`) 위에는 흰색/`--star` 텍스트.
- 본문 텍스트 기본은 `--space` on `--cloud/--sky-soft`(충분한 대비).

---

## 3. 어둠 ↔ 빛 (시맨틱 상태)

색을 "상태"로 쓴다 — 이게 이 디자인의 척추다.

- **DARK (어둠)** = `linear-gradient(180deg, var(--space-2), var(--space), #2A3D7A)` + 별 점들. 사용처: 공동 카운터 디스플레이, 잠든 알, 부화 직전, 프로젝터 송출 기본.
- **LIGHT (빛)** = `--sky-soft` 배경 + `--cloud` 카드. 사용처: 평상시 모든 역할 화면.
- **HATCH FLOOD (깨!)** = 다크에서 → `--sky` 하늘 + `--star` 별빛 글로우 + 컨페티로 **전환**. 알 깨짐 = 어둠이 빛으로.

## 6. 시그니처 — 요시의 알 + 코인 카운터

이 앱이 기억되는 단 하나의 요소. **어둠 속의 알 + 공동 코인 게이지.**

- **알**: 흰/크림 타원 + `--yoshi` 점 + 굵은 `--space` 외곽선, 은은한 글로우. `--space` 다크 필드 위에 둥실 떠 있음(bob).
- **상태 3단**:
  - *잠듦* — 가만히 떠 있는 알(어둠).
  - *금감(진행)* — 코인이 차오를수록 알 표면에 **금(crack)** 이 길어진다(`stroke-dashoffset`를 진행률에 연동). 85% 넘으면 미세 진동.
  - *부화(깨!)* — 100% → 알이 깨지고 `🐣✨` + 빛 블룸 + 컨페티, 화면이 빛으로 전환.
- **코인 게이지**: 골드 줄무늬 채움(`--star`/`--star-deep`), `현재 / 10,000` 큰 숫자(Do Hyeon + Press Start 2P 라벨). 단일 `events.total` 구독 → 모든 화면 동일.

## 7. 컴포넌트 & 토큰

```
--r: 10px;  --r-sm: 7px;  --r-lg: 16px;            /* 모서리 */
--bd: 3px;  --bd-lg: 4px; --bd-color: var(--space);/* 외곽선(친근한 게임 UI) */
--sh:  4px 4px 0 rgba(20,34,77,1);                 /* 하드 그림자(촉각) */
--sh2: 6px 6px 0 rgba(20,34,77,1);
--glow-star: 0 0 24px rgba(246,201,46,.5);
--glow-egg:  0 0 30px rgba(255,224,138,.45);
간격: 4 · 8 · 12 · 16 · 24 · 32 · 48 (px) 스케일
```

- **버튼**: `--bd` 외곽선 + `--r-sm` + `--sh`. 누르면 `translate(2px,2px)` + 그림자 축소(촉각 피드백). 종류: 기본(흰), coin(`--star`), leaf(`--yoshi`), red(`--mario`), ink(`--space`).
- **카드**: `--cloud` + `--bd-lg` 외곽선 + `--r` + `--sh`. 강조 카드는 `--cloud-soft`.
- **배지(미션 타입)**: 경쟁=`--mario`, 정적=보라(`#6b62a8`), 참여=`--yoshi`. Press Start 2P 초소형.
- **리더보드 행**: 순위 칩(`--star`) + 이름 + 진행 바(`--yoshi`) + 코인 숫자(Do Hyeon, `--star-deep`).
- **칩/팀 선택**: 흰 칩 + 선택 시 `--star` 배경.

## 8. 모션

- 알 **bob** 3.6s, 별 미세 트윙클. 진행 시 알 **금 성장**, 85%+ **진동**.
- 코인 총합 **카운트업** 트랜지션. 게이지 채움 0.7s ease.
- **부화**: scale pop + 빛 블룸 + 컨페티(짧게). 버튼 press 마이크로 인터랙션.
- `prefers-reduced-motion`: 모든 애니메이션·트랜지션 **off**, 최종 상태만 표시.

## 9. 아이콘

기능 아이콘은 이모지로(가볍고 명확): 🥚 알 · 🪙 코인 · 🛠 관리자 · 🚩 리더 · 🙌 팀원 · 📖 말씀 · 🎵 찬양 · 🎮 액션 · 🙏 기도 · 🍉 쉼터 · ✨ 빛 · 💥 깨짐 · 🐣 부화 · 🎁 보상. 커스텀이 필요하면 팔레트 색의 **단순 플랫 도형**으로(닌텐도 자산 금지).
